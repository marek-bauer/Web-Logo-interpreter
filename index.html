# Instruction Command
1 cs
Instructions

Moves

forward or fd {steps} - moves turtle forward
backward or bc {steps} - moves turtle backward
left or lt {degrees} - turns turtle to left
right or rt {degrees} - turns turtle to right
pendown or pd - starts drawing line
penup or pu - stops drawing line
setxs {x} {y} - set turtle in position

Pen settings

color {red} {green} {blue} - sets pen color
size {size} - sets pen size in pixels

Mathematical operations

+ / * - ^ - basic operators
(-{number}) - negative number
SIN({degrees}) - sinus
COS({degrees}) - cosine
SQRT({number}) - square root
LN({number}) - natural logarithm
EXP({number}) - powers of e

Logic operations

|| - or
&& - and
! - not

Control flow

repeat {number of repeats} [ {instructions} ]- repeat instructions
if {condition} [ {instructions} ]- run if condition is true
ifelse {condition} [ {instructions} ] [ {instructions} ]- run if condition is true or run another id is false

Function definitions

to {function name} {arguments} {instructions} end - definition
:{name 1} :{name 2} - definition of two arguments
to myfd :n fd :n end - redefinition of fd
to rec :n rec :n - 1 end - definition of recursive function
myfunction {argument} - calling functions
Others
clearscreen or cs - clears screen
